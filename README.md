# EnjoyTrip_

## 프로젝트 목표
- Vue와 Spring boot로 웹 서비스 구현 역량 기르기
- Jwt 적용 및 Redis로 토큰 관리
- home 화면 사진 구성 고민해보기
- AWS 배포 CI/CD
- 여행지 후기 게시판
- 카카오맵 api, 공공데이터 api 이용

## 추가적으로 하고 싶은 것들
- 여행지 선택 및 추천 알고리즘 짜보기
